# lilInspect

Media Metadata Inspector

See what your images, audio, and video files carry: EXIF camera data, GPS location, ID3 tags, MP4 info. Strip it losslessly in the browser.

Part of [Hot App Summer](https://hotappsummer.netlify.app) by [lilAgents](https://lilagents.com): one small, useful web tool shipped every day through June.

## Run it locally

This is an [Astro](https://astro.build) site.

```bash
pnpm install
pnpm dev
```

To build for production:

```bash
pnpm build
```
